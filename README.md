# GameHub E-Commerce System

## Project Overview

GameHub is an online gaming store developed using PHP and MySQL. The system allows users to register, log in, browse gaming products, and make purchases.

## Features

- User Registration
- User Login
- Product Display
- Shopping Cart
- Admin Product Management
- CRUD Operations
- Database Integration

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP
- GitHub

## Database

Database Name:

gamehub

Tables:

- users
- games
- cart
- orders

## Week 6 CRUD Operations

### Create
Administrators can add new products.

### Read
Products are displayed from the database.

### Update
Administrators can edit product details.

### Delete
Administrators can remove products.
